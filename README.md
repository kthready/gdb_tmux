# gdb_tmux
gdb visual tool

Install:
1. Put dbinit to ~/.gdbinit.d/dbinit
2. Put tmux.gdb.conf to ~/.tmux.gdb.conf
3. Install gdb dashboard(https://github.com/cyrus-and/gdb-dashboard):

wget -P ~ https://git.io/.gdbinit

4. Run debug to start
